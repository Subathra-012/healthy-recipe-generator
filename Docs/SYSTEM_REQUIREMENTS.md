1.3.1 HARDWARE REQUIREMENTS
•	Processor Brand: Intel
•	Processor Type: Core i3 or higher
•	Processor Speed: 2 GHz
•	Processor Count: 1
•	RAM Size: 4 GB
•	Memory: DDR3 or later
•	Computer Memory Type: DDR3 SDRAM
•	Hard Drive Size: 500 MB of free disk space
•	Display: 1366 × 768 resolution or higher
•	Peripherals: Keyboard and mouse
•	Internet Connection: Required for Firebase Firestore connectivity, Razorpay payment processing, and real-time WebSocket communication

1.3.2 SOFTWARE REQUIREMENTS
•	Operating System: Windows 10/11, macOS 10.14 or later, modern Linux distributions
•	Application Server: Python FastAPI with Uvicorn (ASGI Server)
•	Front End: React 18, TypeScript, Vite, Tailwind CSS, Radix UI, Framer Motion, Recharts, Lucide Icons
•	Backend Development: Python (version 3.8 or later), FastAPI (0.109+), Pydantic
•	Database: Firebase Firestore (NoSQL, Cloud-hosted)
•	Authentication: Firebase Authentication (Google OAuth, Email/Password)
•	Payment Gateway: Razorpay (Order creation, signature verification, webhooks, refunds)
•	Real-Time Communication: WebSocket (native FastAPI WebSocket support)
•	PDF Generation: ReportLab, QRCode (Python libraries)
•	Email Notifications: SMTP (smtplib with HTML email templates)
•	Security: Google reCAPTCHA v2, PyCryptodome, python-jose (JWT), passlib (bcrypt)
•	Deployment: Google Cloud Run (Serverless auto-scaling), Firebase Hosting (Frontend)
•	Load Testing: Locust Framework
•	Version Control: Git
